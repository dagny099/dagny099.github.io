{%- comment -%} Reusable quote / epigraph component. Single source of truth: _data/quotes.yml Usage ----- Explicit one-off quote: {% include quote.html text="Some line." author="Name" variant="pull" %} Pin a specific quote from a set (relevance-controlled, still single-source): {% include quote.html set="field" id="simon-attention" variant="epigraph" %} Build-time random from a set (changes each deploy): {% include quote.html set="field" variant="epigraph" %} Per-day rotation in the browser (a stable "quote of the day", JS, with the build-time pick as the no-JS fallback): {% include quote.html set="footer" rotate="daily" variant="footer" %} Params ------ text, author : explicit single quote (overrides set) set : key into site.data.quotes (e.g. "footer", "field") id : pin a specific entry from the set by its id pick : "random" (default) or a numeric index, when no id given rotate : "daily" to enable per-day rotation in the browser (needs set) variant : style modifier — "epigraph" | "pull" | "footer" | "plain" class : extra CSS classes {%- endcomment -%} {%- assign q_variant = include.variant | default: "plain" -%} {%- if include.text -%} {%- assign q_text = include.text -%} {%- assign q_author = include.author -%} {%- elsif include.set -%} {%- assign q_list = site.data.quotes[include.set] -%} {%- if include.id -%} {%- assign q_item = q_list | where: "id", include.id | first -%} {%- elsif include.pick and include.pick != "random" -%} {%- assign q_idx = include.pick | plus: 0 -%} {%- assign q_item = q_list[q_idx] -%} {%- else -%} {%- assign q_item = q_list | sample -%} {%- endif -%} {%- assign q_text = q_item.text -%} {%- assign q_author = q_item.author -%} {%- endif -%} {%- if q_text -%} {%- if include.rotate == "daily" and include.set %} {%- endif -%} {%- endif -%}
